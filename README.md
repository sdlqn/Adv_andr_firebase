## Installation
1. Clone the repo
```sh
git clone https://github.com/sdlqn/Adv_andr_firebase
```
2. Install dependencies
3. Connect the app to firebase and setup firebase auth and firestore
4. Run the app (API version 32 or higher is recommended)
